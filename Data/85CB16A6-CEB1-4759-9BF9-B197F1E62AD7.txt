Abstract 
This project proposes in designs a high sensitivity heterodyne interferometer and discusses its accuracy. 
This project uses polarization beam-splitter, the reflector, and matches a piece of half-wave plate, causes the 
dual-frequency laser light beam to have the polarization direction 90 degrees changes, causes its two light way 
the phase difference and the displacement change ratio is traditional heterodyne interferometer's 4 times, so 
may enhance the measurement sensitivity. In the optics precision measurement system design, the heterodyne 
interference technology is an important precision measurement method; Its two laser light source in the entire 
measuring system is in the common-path; According to the above develops the measuring system may have 
the extremely good linearity, the stability and the accuracy, and not easily ambient temperature and outside 
influences. This article proposed that one new measuring method, it uses one HP5529A laser interference 
system, it has the simple establishment experiment construction, and designs by the optical heterodyne 
interferometer's measuring technology, measures its corresponding the displacement change quantity. This 
method has a total optical path structure, it has a high stability, and easy to set up assembly, and a timely 
measure of merit. Stability can be achieved an average error of 0.061, SNR values of up to 43.2db.

B. Encrypting Processes 
Let H be a host binary image and let H* be an overt 
binary image modified from H. The elements of the overt 
image H* contain encrypted codes and the codes are 
classified into five groups of codes. The identification codes 
are used to determine if the codes encrypted in H* use the 
encrypted method proposed in this paper or not; the initial 
position codes are used to assign the initial position of the 
top-left of the sub-divided block; the sub-divided block 
dimension codes are used to indicate the size of the 
sub-divided block; the covert binary image dimension codes 
are used to indicate the size of the covert binary image; (all 
the above-mentioned codes locate at the first row to the 
second one of H*) the information codes are used to decrypt 
covert binary image (locate at the third row to the last one of 
H*).
Identification codes are like passwords and they are used 
to determine whether an overt image contains codes 
proposed in this paper or not. Identification codes are 
composed of 20-bit of pseudo-random binary codes, e.g. 
10011000010000100001. 
Initial position codes are used to assign the initial 
position of the top-left of the sub-divided block and they 
need two sets of 4-bit binary codes. The first set shows the 
number of the row position and the second set shows the 
number of the column position. The codes 0000, 0011, 0111, 
and 1111 are used to represent number of 1, 4, 8, and 16, 
respectively. Other number codes can be used similarly. 
Sub-divided block dimension codes are used to determine 
the size of a sub-divided block and the sub-divided block 
needs two sets of 2-bit binary codes. The first set shows the 
number of rows of the block and the second set shows the 
number of columns of the sub-divided block. The codes 00, 
01, and 11 are used to represent block sizes of 1, 2, and 4, 
respectively. Other dimension codes can be used similarly. 
Covert binary image dimension codes are used to 
determine the size of a covert binary image and the covert 
binary image needs two sets of 6-bit binary codes. The first 
set shows the number of rows of the covert binary image and 
the second set shows the number of columns of the covert 
binary image. The codes 000000, 001111, 011111, and 
111111 are used to represent data sizes of 1, 16, 32, and 64, 
respectively. Other dimension codes can be used similarly. 
Information codes are used to decrypt the covert binary 
image and they contain a lot of codes.  
There are five processes to encrypt a covert binary image 
into the MN host binary image H to derive an MN overt 
binary image H* and they are shown as below. 
(1) Create a 144 binary row string R. The first element 
area (from the left side to the right side) of R is for 
identification codes; the second element area of R is for 
initial position codes, the third element area of R is for 
sub-divided block dimension codes; the fourth element area 
of R is for covert binary image dimension codes. 
(2) The elements of the mn covert binary image D are 
copied to form the elements (from the left side to the right 
side) of a secret data string C with mn elements from the 
first row to the last row (the first priority) and from the left 
side to the right side (the second priority).

Table I shows the comparison the capacity and the PSNR
of the proposed method with Jung and Yooâ€™s method on 
three different host images and the same as sub-divided 
block size. Four host images are Fig. 3 and Fig. 6. The table

but Jung and Yooâ€™s method is dependent of different covert 
images. We found that the overt image and the host image

The proposed encrypting method used here is

explain clearly. Other similar and complex encrypting 
techniques are available and the security of the proposed 
method can be much increased. 
V. CONCLUSION
This paper has exploited a convenient and easy method, 
to realize the data hiding technology for encrypting covert 
binary images in an overt binary image H* modulated from a 
host binary image H. The method is fast calculation, simple, 
increase the storage capacity, and the image quality of the 
overt binary image can be improved. The decoded covert 
images are exactly the same as the original covert images, i.e. 
the decoded covert images do not contain any distortion. It is 
very difficult to decode the hidden covert images from the 
overt image H* for persons without authorization. On the 
other hand, it is easy for authorized users to derive perfect 
covert images. Furthermore, it is not necessary to have the 
host image H or a key for decoding when an authorized user 
decodes all the hidden covert images.
